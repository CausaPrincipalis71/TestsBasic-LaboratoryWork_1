\doxysubsection{Detailed Description}
Класс, создающий поле и перемещающегося по нему юнита 

Юнит может двигаться в 8 направлениях -\/ вправо, влево, вниз, вверх и по-\/диагонали Если поле, на которое юнит пытается подвинуться, не существует, функция возвращает false
